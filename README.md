# GUIProject
A simple Java Swing GUI for simulating a lift bridge control system. Includes controls for bridge movement, car/pedestrian/boat lights, and emergency stop with status display. Prototype only â€” no hardware or network integration.

## How to Run
1. Clone the repo:  
   `git clone https://github.com/huzaifa35120/GUIProject.git`
2. Open `MyFirstWindow.java` in VS Code (or any Java IDE).
3. Compile and run:  
   `javac MyFirstWindow.java`  
   `java MyFirstWindow`
